# Parallel-43

**Practical software for regulated operations — and the building blocks to ship it.**

Parallel-43 is a product studio. We build operational software for businesses that run on regulatory detail — where a missed filing, an unlicensed listing, or a late remittance carries a real cost — along with the developer tools we use to build it. Products are released and opened up over time; this page tracks what's live and what's on the way.

---

## Live now

### ShortletOS — compliance-native short-let management

The permit, guest check-ins, Tourism Dirham, safety requirements, and the paperwork run as one system instead of eight browser tabs. Built for short-term rental operators in **Dubai** (DET Holiday Homes), with **EU** coverage (Regulation 2024/1028) coming next.

**Free tool — Dubai Holiday Home Compliance Checker.** Check your unit against the DET Holiday Homes rules and calculate your monthly Tourism Dirham remittance, entirely in your browser, no signup:

→ **https://shortletos-dubai-compliance.netlify.app**

## How we work

- **Compliance-first.** When the domain is regulated, the rulebook is the specification — not an afterthought bolted on later.
- **Operator-facing.** Plain language, real numbers, and the actual consequence of getting it wrong. No jargon.
- **Focused and reusable.** Small tools that do one job precisely, built on shared pieces we can hand to others.
